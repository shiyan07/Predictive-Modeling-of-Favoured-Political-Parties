# Predictive Modeling of Favoured Political Parties

## Project Overview
This university project builds a predictive model to estimate popular vote share 
in the 2025 Canadian federal election. Using multiple linear regression with 
post-stratification, the model predicts party preference based on survey response data.

## Methods
- Multiple linear regression with post-stratification
- Missing data handling and preprocessing
- Model validation and reproducibility checks
